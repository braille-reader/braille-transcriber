# Training Data

## Included in repo

- `manual/` — Hand-collected Grade 2 braille with English translations (dot notation format)

## Not included (gitignored)

### `brf/` — Bookshare BRF Files
Contracted braille (Grade 2) from professional transcriptions. Obtain BRF files from [Bookshare](https://www.bookshare.org/) for these public domain titles:
- A Christmas Carol — Charles Dickens
- Alice's Adventures in Wonderland — Lewis Carroll
- Moby Dick — Herman Melville
- Pride and Prejudice — Jane Austen
- The Adventures of Sherlock Holmes — Arthur Conan Doyle

### `gutenberg/` — Project Gutenberg Plain Text
Matching English text for alignment with BRF files. Download from:
- https://www.gutenberg.org/cache/epub/24022/pg24022.txt (A Christmas Carol)
- https://www.gutenberg.org/cache/epub/11/pg11.txt (Alice in Wonderland)
- https://www.gutenberg.org/cache/epub/2701/pg2701.txt (Moby Dick)
- https://www.gutenberg.org/cache/epub/1342/pg1342.txt (Pride and Prejudice)
- https://www.gutenberg.org/cache/epub/1661/pg1661.txt (Sherlock Holmes)

### `synthetic/` — Liblouis-Generated Data
Generated by running English text through liblouis with `en-ueb-g2.ctb` table. See `docs/data-pipeline-design.md`.
